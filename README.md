# Rokon's Portfolio

Welcome to my personal portfolio website! This is a modern, responsive portfolio showcasing my projects, skills, and experience as a Full Stack Developer.

## Features

✨ **Modern Design** - Clean and professional UI with gradient colors
📱 **Responsive** - Works seamlessly on desktop, tablet, and mobile devices
⚡ **Fast Performance** - Lightweight and optimized for speed
🔗 **Social Integration** - Links to GitHub, LinkedIn, and other social profiles
📊 **Project Showcase** - Featured projects with descriptions and GitHub links

## Sections

- **Home** - Hero section with call-to-action buttons
- **About** - Brief introduction about me and my expertise
- **Skills** - Categorized skills and technologies I work with
- **Projects** - Featured projects from my GitHub repositories
- **Contact** - Ways to get in touch with me

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- JavaScript (Vanilla JS)
- Font Awesome Icons

## How to Customize

### Update Personal Information

1. **Contact Email**: Open `index.html` and update the email link in the contact section
2. **Social Links**: Update the social media links in the contact section
3. **Projects**: Add or remove project cards in the projects section
4. **Skills**: Update skills in the skills section

### Customize Colors

Edit the color values in `styles.css`:
- Primary gradient: `#667eea` and `#764ba2`
- Accent color: `#ffeb3b`
- Background: `#f8f9fa`

## Deployment

This portfolio is deployed using GitHub Pages. To deploy:

1. Push to your GitHub repository
2. Go to repository settings
3. Enable GitHub Pages from the main branch
4. Your site will be available at `https://yourusername.github.io/portfolio`
